# @txun/video-player

TxunOS ビルトインアプリレイヤー — ビデオプレイヤー。

HTML5 Video API を使用したビデオプレイヤーアプリです。
プレイリスト管理・再生速度変更・Picture-in-Picture・フルスクリーン・キーボードショートカットに対応しています。

## 対応フォーマット

ブラウザの HTML5 Video 実装に依存します（MP4, WebM, Ogg 等）。

## 機能

- 複数ファイルのプレイリスト管理
- シャッフル / リピート（なし / 1本 / 全本）
- 再生速度変更（0.5x 〜 2x）
- Picture-in-Picture (PiP)
- フルスクリーン表示
- キーボードショートカット（Space: 再生/一時停止、←/→: ±5s シーク、↑/↓: 音量）
- ドラッグ&ドロップでファイル追加
